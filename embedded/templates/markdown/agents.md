# Harness CLI Instructions

These instructions are for AI assistants working in this project.

Use the `/hc:start` command when starting a new session to:
- Initialize your developer identity
- Understand current project context
- Read relevant guidelines

Use `@/.harness-cli/` to learn:
- Development workflow (`workflow.md`)
- Project structure guidelines (`spec/`)
- Developer workspace (`workspace/`)

If you're using Codex, project-scoped helpers may also live in:
- `.agents/skills/` for reusable Harness CLI skills
- `.codex/agents/` for optional custom subagents

Keep this managed block so 'harness-cli update' can refresh the instructions.
